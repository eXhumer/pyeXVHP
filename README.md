# eXVHP
Python library for managing videos hosted from various supported video hosting platforms.

## Supported Platforms
* GfyCat
* Imgur
* Streamable
* Streamff
* Streamja

## Package Installation Guide
### Requires
* Python 3.11+
* pip package manager

### Install from source
```console
pip install -e git+https://github.com/eXhumer/pyeXVHP.git@python3#egg=exvhp
```
